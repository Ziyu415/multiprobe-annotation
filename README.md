# MultiProbe Annotation

Annotation interface for the MultiProbe research project (non-commercial academic use).

- Thai: https://ziyu415.github.io/multiprobe-annotation/annotate_th.html
- Vietnamese: https://ziyu415.github.io/multiprobe-annotation/annotate_vi.html

## Data attribution

Images and questions are drawn from the [MTVQA benchmark](https://arxiv.org/abs/2405.11985)
(Tang et al., 2024), licensed under
[CC-BY-NC-4.0](https://creativecommons.org/licenses/by-nc/4.0/).

```
@article{tang2024mtvqa,
  title={MTVQA: Benchmarking Multilingual Text-Centric Visual Question Answering},
  author={Tang, Jingqun and Qi, Qi and Fang, Kun and Shi, Shuyu and Hao, Shaochen
          and Chen, Gang and Xu, Huitao and Yuan, Chunfeng and Li, Weiping and Hu, Bing},
  journal={arXiv preprint arXiv:2405.11985},
  year={2024}
}
```

This repository contains a subset of MTVQA test images (compressed, 800px width,
JPEG quality 50) embedded in a self-contained HTML annotation tool. It is used
solely for research-purpose error-type annotation and will be taken down once
annotation is complete.
